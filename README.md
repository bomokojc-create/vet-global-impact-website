# VET Global Impact Website

Official website for VET Global Impact — a 501(c)(3) veteran-led non-profit empowering U.S. veterans and training youth in the Democratic Republic of Congo through technology education.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Production

```bash
npm run build
npm start
```

## Domain

- Production: https://vetglobalimpact.com
